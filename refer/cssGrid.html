속성 설명
grid grid-template-xxx과 grid-auto-xxx의 단축 속성
grid-area 영역(Area) 이름을 설정하거나, grid-row와 grid-column의 단축 속성
grid-auto-columns 암시적인 그리드 행/열 트랙 크기를 자동으로 설정합니다.
grid-auto-flow 자동 배치 알고리즘이 실행되어 자동으로 배치 되도록 설정합니다.
grid-auto-rows 암시적인 행(Track)의 크기를 정의
grid-column grid-column 속성은 grid-column-start와 grid-column-end 속성을 같이 사용합니다.
grid-column-end 그리드 아이템의 열 끝 위치 지정
grid-column-gap 열과 열 사이의 간격(Line)을 정의
grid-column-start 그리드 아이템의 열 시작 위치 지정
grid-gap xxx-gap의 단축 속성
grid-row grid-row 속성은 grid-row-start와 grid-row-end 속성을 같이 사용합니다.
grid-row-end 그리드 아이템의 행 끝 위치 지정
grid-row-gap 행과 행 사이의 간격(Line)을 정의
grid-row-start 그리드 아이템(Item)의 행 시작 위치 지정
grid-template grid-template-xxx의 단축 속성
grid-template-areas grid-area 속성으로 설정된 그리드 영역의 이름을 참조하여, 그리드 템플릿 영역을 설정합니다.
grid-template-columns grid-template-columns 속성은 가로 컬럼의 크기와 위치를 설정합니다.
grid-template-rows 명시적 행(Track)의 크기를 정의
